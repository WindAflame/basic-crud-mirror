# basic-crud

## Requirements

* Python 3 : <https://www.python.org/downloads/>
* Astral uv : <https://docs.astral.sh/uv/getting-started/>

## Command

|Command|Description|
|---|---|
|install-apt|Installs all the dependencies needed to run the application|
|install|Installs all the libraries needed to run the application|
|freeze|Generates a file that locks the library version|
|dev|Launch application in dev mode|
|start|Launch application in prod mode|
